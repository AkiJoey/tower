# MagicTower

Classical magic tower powered by Swing.

## Version

- `jdk` 11.0.1
- `maven` 3.6.2

## Usage

`$ java -jar MagicTower.jar`

## Control

- [ ↑ ] [ ↓ ] [ ← ] [ → ] Move
- [ SPACE ] Confirm
- [ ESC ] Cancel
- [ S ] Save Archive
- [ A ] Load Archive
- [ R ] Restart Game
- [ Q ] Quit Game
